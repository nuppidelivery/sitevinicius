# Vorté - Agência Premium de Marketing e Tecnologia

Este é o repositório oficial da **Vorté**, uma agência focada no mercado "High Ticket", ajudando empresas a escalarem através de desenvolvimento web, tráfego pago, automação e inteligência artificial.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as melhores e mais modernas ferramentas do ecossistema front-end:

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [Lucide React](https://lucide.dev/)

## 🛠️ Como rodar o projeto localmente

Siga o passo a passo abaixo para testar o site no seu computador:

1. **Instale o Node.js** (versão 18 ou superior).
2. Clone este repositório ou baixe o código-fonte.
3. Abra o terminal na pasta do projeto e instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
5. Abra seu navegador e acesse [http://localhost:3000](http://localhost:3000).

## 🌍 Como fazer o Deploy

Este projeto está pré-configurado para ser publicado na [Vercel](https://vercel.com).
Basta importar este repositório na sua dashboard da Vercel e o deploy acontecerá de forma automática (CI/CD).
